## Run

Clone o projeto

```
git clone https://github.com/ffatique/AAEthSamba24.git
```

Acesse o projeto 

```
cd AAEthSamba24
```

Instale as dependências

```
npm install
```

Copie o conteúdo do arquivo .env.example e crie seu arquivo .env fornecendo as variáveis ​​necessárias.

Inicie o projeto

```
npm run dev
```
